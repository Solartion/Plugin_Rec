# Timelapse Rec — Плагин для Photoshop

Таймлапс-рекордер для Photoshop. Автоматически сохраняет кадры вашего рабочего процесса и собирает их в видео (MP4, MKV, MOV, GIF, WebM).

---

## Установка

### Автоматическая (рекомендуется)

1. Закройте Photoshop
2. Запустите **`install.bat`** от имени администратора
3. Дождитесь завершения — скрипт:
   - Включит режим отладки CEP (нужен для неподписанных расширений)
   - Скачает и установит FFmpeg (≈100 МБ)
   - Создаст ссылку на расширение в папке Adobe CEP
4. Откройте Photoshop → **Window → Extensions → Timelapse Rec**

### Ручная

1. **Режим отладки CEP** — в реестре Windows создайте ключ:
   ```
   HKCU\Software\Adobe\CSXS.9  → PlayerDebugMode = 1
   HKCU\Software\Adobe\CSXS.10 → PlayerDebugMode = 1
   HKCU\Software\Adobe\CSXS.11 → PlayerDebugMode = 1
   ```

2. **FFmpeg** — скачайте с [gyan.dev](https://www.gyan.dev/ffmpeg/builds/) и поместите `ffmpeg.exe` в папку `TimelapseRec\tools\`

3. **Расширение** — скопируйте папку `TimelapseRec` в:
   ```
   %APPDATA%\Adobe\CEP\extensions\
   ```

4. Перезапустите Photoshop → **Window → Extensions → Timelapse Rec**

## Использование

### Начало записи

1. Откройте документ в Photoshop
2. В панели Timelapse Rec укажите **папку для сохранения** (кнопка «Обзор»)
3. Настройте **интервал** между кадрами (по умолчанию 5 секунд)
4. Нажмите **● Запись**

### Во время записи

- Плагин автоматически сохраняет JPEG-копию холста каждые N секунд
- Кадры сохраняются **только при изменениях** — если вы не рисуете, лишние кадры не создаются
- Панель переключается в компактный режим с таймером и счётчиком кадров
- **Пауза** — временно приостанавливает запись
- **Стоп** — останавливает запись и автоматически кодирует видео

### Настройки

| Параметр | Описание |
|----------|----------|
| **Интервал** | Время между кадрами (1–60 сек) |
| **Формат** | MP4, MKV, MOV, GIF или WebM |
| **Удержание кадра** | Длительность показа одного кадра в видео (сек) |
| **Масштаб** | Уменьшает разрешение кадров (1x, 2x, 4x) |
| **Удалить кадры** | Автоматически удалять JPEG-файлы после кодирования |

### Результат

- JPEG-кадры сохраняются в подпапку `rec_YYYYMMDD_HHMMSS` внутри выбранной папки
- После остановки записи FFmpeg автоматически собирает кадры в видеофайл
- Готовое видео появляется в выбранной папке

## Совместимость

- **Photoshop** CC 2019 и новее (версия 20.0+)
- **Windows** 10 / 11
- FFmpeg входит в поставку (скачивается автоматически при установке)

## Решение проблем

**Плагин не появляется в меню Extensions**
→ Убедитесь, что `install.bat` был запущен от имени администратора. Перезапустите Photoshop.

**«FFmpeg not found»**
→ Убедитесь, что файл `ffmpeg.exe` находится в папке `TimelapseRec\tools\`. Или укажите путь вручную через кнопку «Указать».

**Запись не создаёт кадры**
→ Убедитесь, что документ открыт и что вы вносите изменения на холст. Плагин сохраняет кадры только при изменении истории документа.
